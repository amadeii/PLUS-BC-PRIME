# ERP Fiscal SaaS

Sistema **ERP Fiscal SaaS** desenvolvido em **Laravel**, focado em conformidade fiscal brasileira, multiempresa, segurança e escalabilidade.

---

## 📌 Sobre o Projeto

Este projeto é um **ERP Fiscal SaaS** desenvolvido por **Marcos Mello**, com o objetivo de atender empresas de diversos portes, oferecendo controle fiscal, financeiro e operacional em um ambiente seguro e escalável.

O sistema foi projetado para operar em modelo **Software as a Service (SaaS)**, com suporte a múltiplas empresas, usuários, permissões e ambientes fiscais.

---

## 🧾 Principais Funcionalidades

- Multiempresa / Multilocal  
- Controle fiscal completo  
- Emissão de documentos fiscais:
  - NF-e
  - NFC-e
  - CT-e
  - MDF-e
  - NF-Com (quando aplicável)
- Integração com SEFAZ (Homologação e Produção)
- Validação fiscal e regras tributárias
- Controle de usuários e permissões
- Relatórios fiscais e gerenciais
- Dashboard financeiro
- Logs e auditoria
- Controle de licenças SaaS
- API para integrações externas

---

## 🚀 Tecnologias Utilizadas

- **PHP >= 8.1**
- **Laravel >= 10**
- MySQL / MariaDB
- Composer
- Blade / Bootstrap 5
- JavaScript / jQuery / AJAX
- Filas e Jobs (Queues)
- Cache e Sessões
- Cron Jobs
- Certificados digitais A1 / A3

---

## ⚙️ Requisitos do Servidor

- PHP com extensões obrigatórias:
  - OpenSSL
  - PDO
  - Mbstring
  - Tokenizer
  - XML
  - Ctype
  - JSON
  - Curl
- Composer instalado
- MySQL ou MariaDB
- Servidor Apache ou Nginx
- HTTPS habilitado (obrigatório para produção)
